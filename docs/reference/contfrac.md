# `nimblecas.contfrac` — Reference

**Author:** Olumuyiwa Oluwasanmi

Source: `src/contfrac/contfrac.cppm`

Exact continued fractions over the rationals. A **simple continued fraction**
`[a0; a1, a2, …]` encodes a real number as an integer part `a0` plus the
reciprocal of another continued fraction; for a rational number the expansion is
finite and coincides exactly with the trace of the Euclidean algorithm, and for
`sqrt(D)` (non-square positive `D`) it is eventually periodic, `[a0; (a1, …, a_p)]`.
This module supplies both, plus the exact rational **convergents** `p_k/q_k` of
any partial-quotient list, and the **Viskovatov algorithm** that turns a formal
power series into its corresponding continued fraction (C-fraction) — the CF
counterpart of a [Padé](pade.md) approximant. It sits directly above
[`ratpoly`](ratpoly.md).

Everything is **exact**. Simple-CF partial quotients are `int64` integers;
convergents, reconstruction, and the C-fraction coefficients are exact
`Rational`s over `Q`. Following the engine's Rule 32 railway, every fallible step
returns `Result<T>`: an `int64` that would wrap surfaces as
`MathError::overflow`; a perfect-square or non-positive argument to the
quadratic-irrational routine as `MathError::domain_error`; and a Viskovatov pivot
of zero (the C-fraction of that form does not exist) as
`MathError::not_implemented`. Nothing is ever a plausible-looking wrong value.

```cpp
import nimblecas.contfrac;
```

Depends on [`core`](core.md) and [`ratpoly`](ratpoly.md).

## Simple continued fractions of rationals

```cpp
[[nodiscard]] auto from_rational(const Rational& r) -> Result<std::vector<std::int64_t>>;
[[nodiscard]] auto convergents(std::span<const std::int64_t> coeffs) -> Result<std::vector<Rational>>;
[[nodiscard]] auto reconstruct(std::span<const std::int64_t> coeffs) -> Result<Rational>;
```

| Function | Behavior |
| :--- | :--- |
| `from_rational` | The finite simple CF `[a0; a1, a2, …]` of `r` via the Euclidean algorithm. `a0` is signed; every later `a_k` is a positive integer; for a non-integer `r` the last quotient is `>= 2` (the unique **canonical** form). Uses only division/remainder on the already-representable `Rational` operands, so it cannot overflow. |
| `convergents` | The convergents `p_0/q_0, p_1/q_1, …` via `p_k = a_k p_{k-1} + p_{k-2}`, `q_k = a_k q_{k-1} + q_{k-2}` (`p_{-1}=1, p_{-2}=0, q_{-1}=0, q_{-2}=1`), exact over `Q`. Each returned `Rational` is the reduced convergent; since consecutive CF convergents are coprime, its `numerator()`/`denominator()` are exactly `p_k`/`q_k`. |
| `reconstruct` | The exact `Rational` value of the finite CF, folded bottom-up: `value = a_n`, then `value = a_k + 1/value`. For a list from `from_rational` this **round-trips** to the original `r`. |

**Round-trip guarantee.** `reconstruct(from_rational(r)) == r` for every
`Rational r`, and the last element of `convergents(from_rational(r))` equals `r`.

## Periodic continued fractions of quadratic irrationals

```cpp
struct PeriodicCF {
    std::vector<std::int64_t> prefix;  // acyclic part (for sqrt(D): a single element [a0])
    std::vector<std::int64_t> period;  // the repeating block
};

[[nodiscard]] auto quadratic_irrational_cf(std::int64_t D) -> Result<PeriodicCF>;
```

The CF of `sqrt(D)` for a positive non-square integer `D`, generated by the
canonical integer recurrence

```
m_{k+1} = d_k a_k - m_k
d_{k+1} = (D - m_{k+1}^2) / d_k          (exact: d_k always divides D - m_{k+1}^2)
a_{k+1} = floor((a0 + m_{k+1}) / d_{k+1})
```

starting from `(m0, d0, a0) = (0, 1, floor(sqrt D))`. The acyclic `prefix` is the
single term `[a0]`; the `period` is the repeating block `a1 … a_p`, which closes
exactly when `a_k` first reaches `2 a0` (its canonical last element). Every
quantity stays a bounded exact integer, and each product is overflow-checked.

## Viskovatov: series → corresponding continued fraction

```cpp
struct SeriesCF {
    Rational b0;              // constant term b0 = c_0
    std::vector<Rational> a; // partial numerators a_1, a_2, … (stage k is a_k*x / (1 + …))
};

[[nodiscard]] auto viskovatov(std::span<const Rational> series_coeffs) -> Result<SeriesCF>;
```

Given `f(x) = c_0 + c_1 x + c_2 x^2 + …`, the Viskovatov algorithm produces the
regular C-fraction (Thiele-style) form

```
f(x) = b0 + (a_1 x)/(1 + (a_2 x)/(1 + (a_3 x)/(1 + …)))
```

with `b0 = c_0`. It maintains the current tail of `f` as an exact ratio
`N(x)/D(x)` of two truncated series (initially `(f - c_0)/1`); at each stage it
divides out one factor of `x`, reads off `a_k = [x^0] shift(N) / [x^0] D`, and
updates `(N, D) <- (a_k D - shift(N), shift(N))`. The `k`-th two-term convergent
of the C-fraction is exactly the `[ceil(k/2) / floor(k/2)]` **Padé** approximant
along the staircase.

### Honesty boundary

The result is **exact for the supplied (truncated) coefficients**, and only
partial numerators fully determined by the input are returned — an input of `L`
coefficients yields `b0` together with `a_1 … a_{L-1}`, and none are fabricated
past the data (the working reliability count shrinks by one per stage so no
coefficient tainted by the unknown next term is ever read). The expansion **stops
early and exactly** when the tail numerator vanishes identically — then the finite
C-fraction reproduces `f` as a rational function (e.g. `{1,1,1,1}` →
`(1; 1, -1)`, and `1 + x/(1-x) = 1/(1-x)`). If a required pivot is zero while the
numerator is not identically zero, the regular C-fraction of this form **does not
exist** (e.g. `1 + x^2`, whose first non-constant term is `x^2`): this is reported
honestly as `MathError::not_implemented`, never a wrong expansion.

## Error model

| Condition | Error |
| :--- | :--- |
| `convergents` / `reconstruct` of an empty coefficient list | `MathError::domain_error` |
| `reconstruct` hits a zero intermediate denominator (only from a non-canonical, hand-built list) | `MathError::division_by_zero` |
| `quadratic_irrational_cf(D)` with `D <= 0` or `D` a perfect square | `MathError::domain_error` |
| `viskovatov` of an empty coefficient list | `MathError::domain_error` |
| `viskovatov` pivot is zero with a non-vanishing numerator (C-fraction does not exist) | `MathError::not_implemented` |
| Any `int64` numerator/denominator computation wraps during the exact arithmetic | `MathError::overflow` |

## Worked examples

```cpp
import nimblecas.contfrac;
import nimblecas.ratpoly;
using namespace nimblecas;

auto ri  = [](std::int64_t v) { return Rational::from_int(v); };
auto rat = [](std::int64_t n, std::int64_t d) { return Rational::make(n, d).value(); };

// Simple CF of 355/113 and its round-trip.
auto cf = from_rational(rat(355, 113)).value();     // [3, 7, 16]
reconstruct(std::span<const std::int64_t>{cf});     // 355/113
convergents(std::span<const std::int64_t>{cf});     // 3/1, 22/7, 355/113

// sqrt(2) = [1; (2)]; convergents obey the Pell identity p^2 - 2 q^2 = +-1.
auto q2 = quadratic_irrational_cf(2).value();        // prefix [1], period [2]

// sqrt(23) = [4; (1, 3, 1, 8)].
auto q23 = quadratic_irrational_cf(23).value();      // prefix [4], period [1,3,1,8]
quadratic_irrational_cf(9).error();                  // domain_error (perfect square)

// Viskovatov of the geometric series 1/(1-x): terminates exactly.
auto g = std::vector<Rational>{ri(1), ri(1), ri(1), ri(1)};
auto cg = viskovatov(std::span<const Rational>{g}).value();   // b0 = 1, a = [1, -1]

// Viskovatov of exp: (1; 1, -1/2, 1/6); its [1/1] convergent is the [1/1] Pade of exp.
auto e = std::vector<Rational>{ri(1), ri(1), rat(1, 2), rat(1, 6)};
viskovatov(std::span<const Rational>{e}).value();    // b0 = 1, a = [1, -1/2, 1/6]

// Honest breakdown: 1 + x^2 has no regular C-fraction of this form.
auto bad = std::vector<Rational>{ri(1), ri(0), ri(1)};
viskovatov(std::span<const Rational>{bad}).error();  // not_implemented
```
